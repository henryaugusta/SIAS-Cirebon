## Installation
- [Download the latest release](https://github.com/itherdevs/SIAS/archive/master.zip).
or clone the repo :
```bash
git clone https://github.com/itherdevs/SIAS.git
```

And don`t forget to install the package dependencies with command :
```bash
composer install
```
